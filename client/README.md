## Contributing 🤜🏼🤛🏼

  - Please do `npm install` after clone 👥
  - To start the client, do `npm run start` 🏃🏽‍♂️
  - Finished yet? do `npm run build` to get the production app 👷🏼‍♀️
  - Have fun ! 🤩

## Depedencies 🧐

  - React ⚛️
  - Material UI 👾
  - Lodash 👌🏽
  - Framer Motion 🚶🏼‍♂️
  - Sass 🎨
  - and many more.... 😎
